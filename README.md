# Curso-finazas-rpg

Bienvenido a **Curso Finanzas rpg**, un proyecto educativo diseñado para enseñar Finanzas Personales y Empresariales desde cero mediante una experiencia tipo RPG.

## Objetivos

- Comprender el origen del dinero.
- Dominar las finanzas personales.
- Aprender contabilidad y economía.
- Entender el sistema fiscal mexicano (SAT).
- Invertir de forma responsable.
- Crear y escalar empresas.
- Alcanzar la libertad financiera.
- Construir patrimonio generacional.

## Estructura

- Prólogo
- Nivel 0: El origen del dinero
- Nivel 1: Cómo funciona el dinero
- Nivel 2: Sistema financiero mundial
- …
- Nivel 12: Patrimonio generacional

Cada nivel incluirá:
- Historia
- Explicación sencilla
- Explicación profesional
- Casos reales
- Ejercicios
- Mini examen
- Proyecto
- Misiones y logros
Curso interactivo de Finanzas Personales y Empresariales en Markdown
